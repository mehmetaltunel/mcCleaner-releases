## Features

### Cache Cleaner
Scans and removes application cache files. On macOS it checks `~/Library/Caches`, on Windows it checks `%LOCALAPPDATA%\Temp`. Only third-party app caches are listed — system-critical caches are always protected.

### Log Cleaner
Finds old log and diagnostic report files across the system. Helps free up space from crash logs, app logs, and system diagnostic reports.

### Privacy Cleaner
Detects browser data (cache, cookies, browsing history, session data) for all major browsers: Chrome, Safari, Firefox, Edge, Brave, Arc, and Opera. You can selectively clean any category per browser.

### DevCleaner
Scans cache directories for 19+ developer tools: npm, Yarn, pnpm, pip, Conda, RubyGems, CocoaPods, Cargo, Go, Gradle, Maven, NuGet, Homebrew, Docker, Xcode, Flutter/Dart, Composer, Android SDK, and Visual Studio. Includes an interactive file browser to inspect and selectively delete contents.

### System Health Check
Real-time monitoring dashboard showing CPU usage, RAM, disk space, battery status, CPU/GPU temperatures, network traffic, load average, process count, and uptime. Displays an overall health score.

### Disk Analyzer
Visual disk usage overview with a large file finder. Streams results in real-time and supports scan cancellation.

### App Manager
Lists all installed applications with their sizes and icons. Scans for residual files left behind by apps. Supports complete uninstallation — moves the app and all its residuals to Trash.

## Ozellikler

### Onbellek Temizleyici
Uygulama onbellek dosyalarini tarar ve temizler. macOS'ta `~/Library/Caches`, Windows'ta `%LOCALAPPDATA%\Temp` dizinlerini kontrol eder. Sadece ucuncu parti uygulama onbellekleri listelenir — sistem onbellekleri her zaman korunur.

### Log Temizleyici
Sistemdeki eski log ve tani raporlarini bulur. Crash loglari, uygulama loglari ve sistem tani raporlarindan yer acmanizi saglar.

### Gizlilik Temizleyici
Chrome, Safari, Firefox, Edge, Brave, Arc ve Opera dahil tum buyuk tarayicilarin verilerini (onbellek, cerezler, gecmis, oturum verileri) tespit eder. Her tarayici icin istediginiz kategoriyi secip temizleyebilirsiniz.

### Gelistirici Araclari Temizleyici
npm, Yarn, pnpm, pip, Conda, RubyGems, CocoaPods, Cargo, Go, Gradle, Maven, NuGet, Homebrew, Docker, Xcode, Flutter/Dart, Composer, Android SDK ve Visual Studio dahil 19'dan fazla gelistirici aracinin onbellek dizinlerini tarar. Etkilesimli dosya tarayicisi ile iceriklerinizi inceleyip secici olarak silebilirsiniz.

### Sistem Saglik Kontrolu
CPU kullanimi, RAM, disk alani, batarya durumu, CPU/GPU sicakliklari, ag trafigi, sistem yuku, islem sayisi ve calisma suresini gosteren gercek zamanli izleme paneli. Genel bir saglik puani goruntulenir.

### Disk Analizoru
Buyuk dosya bulucu ile gorsel disk kullanim analizi. Sonuclari gercek zamanli olarak aktarir ve taramayi iptal etmeyi destekler.

### Uygulama Yoneticisi
Tum yuklu uygulamalari boyut ve ikonlariyla listeler. Uygulamalardan kalan artik dosyalari tarar. Tam kaldirma destegi — uygulamayi ve tum artik dosyalarini Cop Kutusuna tasir.
